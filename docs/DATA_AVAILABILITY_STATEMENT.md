# Data availability statement

All source code, workload generators, scheduler implementations, configuration scripts, processed tables, generated figures, and raw measurement files used for the manuscript are included in this repository.

Before journal submission, replace the placeholder repository URL and DOI with:

- GitHub repository: `GITHUB_RELEASE_URL`
- Archived release DOI: `ZENODO_DOI`

The repository contains the formal supply-chain workload matrix, supporting robustness and ablation experiments, scheduler-overhead measurements, and the manuscript source used to build the reviewed PDF. Runtime logs are not required for the reported tables and are excluded where redundant; the retained JSON and CSV files contain the client-side and scheduler-side metrics used for analysis.

# Release notes: v1.0.0-peerj-submission

This release is intended as the archival reproducibility package for the PeerJ Computer Science submission.

Included materials:

- CAPS scheduler, Traditional Lock, ED-MVCC-Reimpl, DRT-Reimpl, and Native Fabric gateway scripts.
- Hyperledger Fabric test-network adaptations used for the 4-peer, 1-Raft-orderer deployment.
- Supply-chain chaincode and workload-generation scripts.
- Raw JSON/CSV metrics for the formal 10-repeat experiments and supporting analyses.
- Processed LaTeX table rows and manuscript figures.
- PeerJ manuscript source and a reviewed local PDF build.
